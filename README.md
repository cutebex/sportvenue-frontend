## Installations

### Clone

- Clone this project to your local machine `https://github.com/cutebex/sportvenue-react.git`

### Setup
  > In the root directory, run the command
  ```shell
  $ npm install
  ```

### Setup
  > In the root directory, run the command
  ```shell
  $ npm run start
  ```


  - Use `http://localhost:3000` as base url for endpoints
